# AdaptiveOS Landing Page

This repository contains the source code for the official [AdaptiveOS](https://adaptive-os.github.io/) landing page.

## Overview

AdaptiveOS is a performance-focused, Vanilla GNOME experience built on top of Ubuntu. This website serves as the primary hub for project information and ISO distribution.

## Tech Stack

*   **HTML5**
*   **Tailwind CSS** (via CDN)
